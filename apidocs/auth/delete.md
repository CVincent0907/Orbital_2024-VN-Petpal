# Delete Account

Delete the account of the logged in request.

**URL** : `/api/auth/delete/`

**Method** : `DELETE`

**Auth required** : YES

**Permissions required** : None

**Data** : `{}`

## Response

**Code** : `204 NO CONTENT`

**Content** : `{}`

## Notes

* Will remove all pets listed by this account if it is a shelter.
